הנה ריכוז מלא של כל קודי מזג האוויר (Present Weather) לפי ICAO, כפי שמשמשים ב-METAR/ATIS:

---

## קודי מזג האוויר ב-ATIS / METAR

### עוצמה (Intensity Prefix)

| קוד | משמעות |
|-----|--------|
| `-` | קל (Light) |
| *(ללא קידומת)* | בינוני (Moderate) |
| `+` | חזק (Heavy) |
| `VC` | בסביבה (In Vicinity, 5–10 מייל) |

---

### מתארים (Descriptors)

| קוד | משמעות |
|-----|--------|
| `MI` | רדוד (Shallow) |
| `PR` | חלקי (Partial) |
| `BC` | טלאים (Patches) |
| `DR` | נסחף נמוך (Low Drifting) |
| `BL` | מתנפץ/מתנשא (Blowing) |
| `SH` | מקלחות (Showers) |
| `TS` | סופת רעמים (Thunderstorm) |
| `FZ` | קופא (Freezing) |

---

### משקעים (Precipitation)

| קוד | משמעות |
|-----|--------|
| `DZ` | טפטוף (Drizzle) |
| `RA` | גשם (Rain) |
| `SN` | שלג (Snow) |
| `SG` | גרגירי שלג (Snow Grains) |
| `IC` | גבישי קרח (Ice Crystals) |
| `PE` | כדורי קרח (Ice Pellets) |
| `GR` | ברד (Hail) |
| `GS` | ברד קטן / גלושות שלג (Small Hail / Snow Pellets) |

---

### ערפול וראות (Obscurations)

| קוד | משמעות |
|-----|--------|
| `FG` | ערפל (Fog) — ראות מתחת ל-1000 מטר |
| `BR` | אד (Mist) — ראות 1000–5000 מטר |
| `HZ` | אובך (Haze) |
| `FU` | עשן (Smoke) |
| `DU` | אבק (Dust) |
| `SA` | חול (Sand) |
| `VA` | אפר וולקני (Volcanic Ash) |
| `PY` | ריסוס ים (Spray) |

---

### תופעות אחרות (Other Phenomena)

| קוד | משמעות |
|-----|--------|
| `PO` | סופת אבק/חול קטנה (Dust/Sand Whirls) |
| `SQ` | סקוול (Squalls) |
| `FC` | עמוד אוויר / טורנדו (Funnel Cloud / Tornado) |
| `SS` | סופת חול (Sandstorm) |
| `DS` | סופת אבק (Duststorm) |
| `BLSN` | שלג מתנשא (Blowing Snow) |
| `BLDU` | אבק מתנשא (Blowing Dust) |
| `BLSA` | חול מתנשא (Blowing Sand) |

---

### שילובים נפוצים

| קוד | משמעות |
|-----|--------|
| `TSRA` | סופת רעמים עם גשם |
| `+TSRA` | סופת רעמים עם גשם כבד |
| `SHRA` | מקלחות גשם |
| `SHSN` | מקלחות שלג |
| `FZRA` | גשם קופא |
| `FZDZ` | טפטוף קופא |
| `FZFG` | ערפל קופא |
| `-RASN` | גשם ושלג קל |
| `BCFG` | ערפל טלאים |
| `MIFG` | ערפל רדוד |
| `PRFG` | ערפל חלקי |
| `VCSH` | מקלחות בסביבה |
| `VCTS` | סופת רעמים בסביבה |
